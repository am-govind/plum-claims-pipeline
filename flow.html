1 The shape (Hexagonal layers)

The system is organised into four layers. Dependencies point inward: interfaces talks to application talks to domain. infrastructure sits on the outside and implements the abstract ports the application defines.

2 Ports & Adapters (the abstraction line)

Every external concern is a small interface (port) the application owns, and a concrete adapter in infrastructure.

Port (application)Adapter (infrastructure)What it abstracts
ClaimRepositorySqlAlchemyClaimRepositorysave / get / list_recent on ClaimState
PolicyRepositoryJsonPolicyRepositoryLoad the active PolicyTerms from policy_terms.json
RuleEngineDslRuleEngine (lives in domain)Evaluate policy_rules.json against a ClaimState
LLMProviderMockProvider / GeminiProviderExtract one document → ExtractedDocument + LLMUsage
EventBusInMemoryEventBus + handlersPublish DomainEvents drained from ClaimState

4 The pipeline (LangGraph)

A ClaimState flows through the agents with two deliberation loops. 

flowchart TD
  S(["submit"]):::start
  I["Intake"]:::critical
  IStop["stop (early_stop)"]:::stop
  DV["Document Verification"]:::critical
  DVStop["stop (early_stop)"]:::stop
  EX["Extraction (llm_provider)"]:::soft
  RV["re_verification (1 retry)"]:::delib
  CD["Contradiction Detection"]:::soft
  PA["Policy Adjudication (rule_engine)"]:::soft
  FC["Financial Calculation"]:::soft
  FD["Fraud Detection"]:::soft
  PR["policy_reconsider (1 retry)"]:::delib
  DS["Decision Synthesizer"]:::critical
  END(["decision"]):::start
  S --> I
  I -- "early_stop" --> IStop
  I -- ok --> DV
  DV -- "early_stop" --> DVStop
  DV -- ok --> EX
  EX -- "low conf / issues" --> RV
  RV -- "(cap = 1)" --> CD
  EX -- ok --> CD
  CD --> PA
  PA --> FC
  FC --> FD
  FD -- "fraud + clean policy" --> PR
  PR -- "(cap = 1)" --> DS
  FD -- ok --> DS
  DS --> END

7 What this buys you

  • Swapability.Replace JSON policy storage with a REST policy service: write a new PolicyRepository adapter, change one line in composition.py.
  • Testability.Unit tests build the exact adapter they need and inject it. No global state to reset.
  • One source of truth.compose() is the only function that knows the wiring.
  • No hidden state.Zero lru_cache / module-globals. Lifetimes are explicit and owned by the Container.
Diagrams reflect the live code in backend/app/. Pipeline wiring lives in backend/app/application/pipeline.py; the composition root is backend/app/composition.py. For per-agent contracts see AGENTS.md; for component IO/error contracts see docs/COMPONENT_CONTRACTS.md.
